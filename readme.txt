Transform your website into a digital business card hub with an elegant floating QR code that allows instant contact sharing.

== Description ==

My VCard Plugin is a powerful tool that modernizes how you share contact information through your WordPress website. With a sleek floating button and QR code interface, visitors can instantly save your contact details to their smartphones.

= Key Features =

* **Floating QR Code Button**: A customizable, non-intrusive button that follows your visitors as they browse
* **Instant Contact Sharing**: Generate professional VCard (.vcf) files compatible with all major contact applications
* **Customizable Positioning**: Place the floating button where it works best for your site - bottom left, center, or right
* **Comprehensive Contact Fields**: Include all essential contact information:
  * Company name
  * First and last name
  * Mobile and landline numbers
  * Website URL
  * Email address
  * Complete postal address
  * Job title
  * Custom QR code text

= Perfect For =

* Professionals and freelancers
* Business websites
* Personal portfolios
* Contact pages
* Event landing pages
* Digital business cards

= How It Works =

1. Install and activate the plugin
2. Fill in your contact information in the WordPress admin panel
3. Choose your preferred button position
4. A floating button appears on your website
5. Visitors click the button to reveal a QR code
6. Scanning the QR code instantly downloads your contact information to their device

= Technical Features =

* Clean, optimized code
* Mobile-responsive design
* Lightweight implementation
* Secure data handling
* Easy WordPress integration
* Custom endpoint for VCard downloads

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/floating-qr-code-vcard` directory, or install the plugin through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->VCard screen to configure your contact information

== Frequently Asked Questions ==

= Is the QR code generator reliable? =
Yes, we use a trusted QR code generation API that ensures high-quality, scannable codes.

= Can visitors download the contact information without a QR code? =
Yes, the VCard file can be accessed directly through a URL if needed.

= Is the plugin GDPR compliant? =
Yes, the plugin only displays the contact information you explicitly provide in the settings.
